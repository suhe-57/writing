# 检查清单

---

## 开题调研（场景 A）

开题保持完整检查流程。每阶段结束可对照做缺口扫描，不足则补问。**言情类型**按下列清单执行；**古代/悬疑/科幻等类型**可将阶段 2、3、5 的言情术语替换为对应类型术语（如双主角→核心人物，男主/女主→主角A/对手或侦探/嫌疑人等），追问维度按类型微调。

### 阶段 1 项目基线
- [ ] 题材/类型明确
- [ ] 时代背景明确
- [ ] 篇幅预期或受众有提及
- [ ] 情绪基调有描述
- [ ] 结局走向有倾向（HE/BE/开放）

### 阶段 2 核心人物设定（言情：双主角）
- [ ] 主角目标或动力明确
- [ ] 主角缺陷或成长点明确
- [ ] 对手/搭档（言情：女主）的立场、防线或顾虑明确
- [ ] 对手/搭档（言情：女主）的需求或软肋明确
- [ ] 核心人物初始关系或第一印象有描述

### 阶段 3 主线机制
- [ ] 主线结构清晰（言情：旅途/邂逅线；悬疑：案件线；科幻：核心矛盾线等）
- [ ] 至少 1 个关键转折/邂逅/冲突节点明确
- [ ] 人物关系或局势的推进与回撤有大致方向

### 阶段 4 冲突系统
- [ ] 至少 1 类外部阻碍或内部误解明确
- [ ] 若有价值观/立场冲突，已点出
- [ ] 至少 1 个临界事件有概念

### 阶段 5 弧光与结局（言情：双主角情感弧光）
- [ ] 主角成长/转变的证据链有 1–2 个关键节点（言情：男主「俘获芳心」、女主「陷入恋情」）
- [ ] 对手/搭档的相应变化有 1–2 步
- [ ] 结局或余韵有描述

### 阶段 6 一致性校验
- [ ] 时间线可执行
- [ ] 人设前后一致
- [ ] 动机闭环

**开题出稿条件**：以上各阶段核心项均勾选，因果链完整，无未决逻辑冲突。

---

## 章节调研快线（续写 / 改写 / 章节规划）

续写、改写、章节规划快线共用此清单。只保留最必要的 5 项，避免过度追问。

### 快线清单（5 项）
- [ ] **功能**：这几章/本章在整体中的功能是什么？（推进主线 / 情绪转折 / 支线收束等）
- [ ] **人物位置**：主角此刻的情绪和关系位置是什么？与前文衔接到哪一步？
- [ ] **承上**：与前文怎样衔接？有哪些不能推翻的设定或基调？
- [ ] **启下**：这几章改完后，下一章/下一段该接什么？
- [ ] **不可动**：有哪些设定、人物、时间线不能碰？

满足以上 5 项后，可输出短版章节方案（每章标题 + 200–300 字内容介绍）。用户说「展开」时再补长报告。

---

## 改写前置必问（场景 C 专用）

改写前**必须**先向用户确认以下 3 点，不得跳过：

1. **保留项**：哪些内容你想保留？
2. **不满项**：哪些你最不满意？
3. **幅度**：你要小修、中调，还是重构？

确认后再给章节方案。
