# 输出模板（与 dagang 衔接）

以下模板用于不同场景。**开题默认长报告；续写、改写默认短版**。用户说「展开/详细报告/完整分析」时，才用长版。

---

## 输出 1：开题——长报告（默认）

开题调研默认输出完整报告。言情类型用「双主角关系发展主线」；古代/悬疑/科幻等类型可替换为「核心人物/主线发展」（如侦探与案件线、主角与对手线等）。

```markdown
# [书名/暂定名] 开题调研报告

## 创作目标与主题陈述
- 核心主题：
- 情绪基调：
- 目标受众与篇幅预期：
- 结局走向（HE/BE/开放式）：

## 核心主线发展（按阶段）（言情：双主角关系发展主线）
- 阶段一（开局/邂逅）：
- 阶段二（推进/试探）：
- 阶段三（升温/回撤）：
- 阶段四（危机/临界）：
- 阶段五（确认/结局）：

## 分幕式总体大纲（关键情节点）
- 第一幕：
- 第二幕：
- 第三幕：

## 时间线草图（里程碑事件）
- 事件1（时间/场景）：
- 事件2：
- …

## 人物特征草图（可映射到 characters）（言情：男主/女主；其他类型：主角/对手或核心人物）
- 主角/男主：目标/缺陷/弧光要点
- 对手/女主：防线/需求/弧光要点
- 重要配角（若有）：

## 风险与待确认项
- 待用户确认或下一轮追问入口：
```

## 输出 2：续写——短版（默认）

续写调研**默认用此短版**。每章只给「标题 + 200–300 字内容介绍」。

```markdown
# [项目名] 后续章节梳理

## 我的判断（最多 3 条）
1. 
2. 
3. 

## 建议方向
（1 条主路线，1–2 句）

## 章节简版方案

### 第 X 章《标题》
（200–300 字：本章讲什么、核心事件、情绪落点、与前后章衔接）

### 第 X+1 章《标题》
（同上）

### …

---
如需展开成完整报告，请说「展开」。
```

## 输出 4：续写 / 改写——长版（仅当用户说「展开」时）

用户明确说「展开/详细报告/完整分析」时使用。

**续写长版**可含：当前阶段诊断、多路线对比表、推荐路线、未来 3–5 章可写粒度、节奏与冲突建议。

**改写长版**可含：问题清单、原因拆解、保守/平衡/激进三档方案、推荐方案与改动要点、验收清单。

落盘时按对应场景命名：续写策略报告、改写策略报告。

---

## 使用说明

- 完成调研后，按场景选用上述模板。
- 续写、改写、章节规划快线：**默认短版**，直接给章节方案。
- 开题：默认长报告。
- 落盘目录：`reports/`，命名规则见 SKILL.md §6。
